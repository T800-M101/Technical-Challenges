# Angular Challenges Portfolio

A collection of solved and documented Angular technical challenges. Each challenge includes a working demo, source code, and an explanation of the technical decisions made.

## 🔗 Live Demo

[View portfolio](https://oh-my-portfolio.netlify.app/)

## 📦 Challenges Included

| Challenge | Difficulty | Tags |
|---|---|---|
| Reactive Form with Custom Validations  | Intermediate | Forms, Validation, RxJS |
| API Consumption with RxJS | Intermediate | HTTP, RxJS, API |
| Auth Guard + Route Protection | Advanced | Router, Guards, Auth |
| UI Components Library | Advanced | Components, Library, CSS |

## 🛠 Stack

- Angular 19+ (Standalone Components)
- Tailwind CSS
- RxJS
- TypeScript strict mode

## 🚀 Run Locally

```bash
npm install
ng serve
```

## 🏗 Architecture

```
src/app/
├── core/           # models and data
├── shared/         # shared components
└── features/       # each challenge is a lazy-loaded feature module 
```
